## Customising your ankiChess Notes

This addon can be customised to allow for many different types of cards. This includes options such as configuring the timer, disabling Variation lines and many more. Config options can be saved on a per note basis (Or card if you want to make multi sided notes). 

Using Anki's built in tools for managing note types you can clone this template to have "ankiChess", "chess flipped", "chess Study" etc. Refer to the "Interactive Examples" section in the "Help Menu" menu in the template for possible setup examples. 

### Puzzle
Default setup, play a tactic style card according to a given PGN.

### Flipped Puzzle
Anki Template > Flip PGN
Auto plays First move of a PGN and orientates from seconds players perspective. This is the default setup for Lichess Puzzles.

### Study Mode
Anki Template > Play Both Sides
Same as Puzzle but requires user to play Both sides of a PGN. Useful for studying games. 

### Play vs Engine
You can make am engine card by putting a FEN instead of a PGN into the anki PGN field. This will Load that position and have you Play vs an engine. Refer to AI settings for controlling options like strength. 

The anki chess [Companion Addon](https://ankiweb.net/shared/info/1300975327) will handle auto updating multiple note types.
